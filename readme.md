### Clone
```
$ git clone https://github.com/MQTEnt/laravel-reactjs-crud.git lareact
```
### Config your database in .env file

### Migrate database
```
$ php artisan migrate
```
### Composer update
```
$ cd lareact
$ composer update
```

### NPM install
```
$ npm install
````

### Build bundle.js
```
webpack
```

### Run Server
```
$ php artisan serve
```

### Open localhost:8000
